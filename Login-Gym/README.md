# SIMPLE login Web Application 

## How To Run :
- windows:

        py -3 -m venv venv  
        .\venv\Scripts\activate.bat

- Linux or Mac:

        python3 -m venv venv
        . venv\bin\activate


## Then Install requirements Library's:
        
        pip install -r requirements.txt 

___

## How Run Local Host : 
 
 - Windows :        
        
        set FLASK_ENV=development
        flask run

- Linux Or Mac:

        export FLASK_ENV=development
        flask run



### and then just type: flask run
